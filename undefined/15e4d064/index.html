JSON学习笔记

1 创建JSON对象

JSON 是一种数据格式，并不是编程语言

1.1 JSON语法

相比于XML ，JSON格式最大的优点是易于人的阅读和编写，通常不需要特殊的工具，就能读懂和修改，是一种轻量级的数据交换格式。

JSON文件都是被包裹在一个大括号中 {}，通过key:value的方式来表达数据。看起来同 JavaScript 的对象表达方式十分相似，但是有所不同：JSON的Key必须包裹在一个双引号中，在实践中，编写 JSON 的时候，忘了给 Key 值加双引号或者是把双引号写成单引号是常见错误。

JSON 文件中无法使用注释，试图添加注释将会引发报错。

1.2 JSON的数据格式

  1. 数字，包含浮点数和整数
  2. 字符串，需要包裹在双引号中
  3. Bool值，true 或者 false
  4. 数组，需要包裹在方括号中 []
  5. 对象，需要包裹在大括号中 {}
  6. Null

其他任何格式都会触发报错
