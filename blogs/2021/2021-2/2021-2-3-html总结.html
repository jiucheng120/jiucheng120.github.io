  html总结

一、基础知识

  1. <html> </html> 根标签，页面中最大的标签
  2. <head> </head> 文档的头部
  3. <title> </title> 文档的标题
  4. <body> </body> 文档的主体
  5. <br/> 单标签（少见）
  6. ctrl S， ctrl+"+" ，Alt+"B"

二、必须写的三个代码

  1. <!DOCTYPE> 采取HTML5版本，文档类型声明标签，不属于HTML标签
  2. <html lang="en"> 表示英文网站，zh-CN表示中文网站，fr表示法语网站，但不影响所有语言表示
  3. <meta char set="UTF-8"> 字符集是多个字符的集合，以便计算机能够识别和存储各种文字

三、常用标签

  1. 标题标签：
    <h1>-<h6>根据重要性递减，每个标题独占一行
  2. 段落标签：
    <p></p>
    1. 段落之间有空隙
    2. 根据网页大小换行
  3. 换行标签：
    <br>或 <br />表示打断，换行
  4. 文本格式化标签
    1. <strong></strong>语义更强烈,<b></b>
    2. <em></em>语义更强烈,<i></i>
    3. <del></del>语义更强烈,<s></s>
    4. <ins></ins>语义更强烈,<u></u>
  5. <div><span>无语义的盒子
    1. <div>表示分割，分区，用于布局，单独占一行。大盒子
    2. <span>表示跨度跨距，一行可以放多个。小盒子
  6. 图像标签
    <img src="" />单标签，必须将被插入的文件和html文件放到一个目录下
    1. src用于指定图像文件的路径和文件名
    2. alt替换文本。图像不能显示的文字
    3. title提示文本。鼠标放到图像上显示的文字
    4. width设置图像的宽度
    5. height设置图像的高度，宽度高度修改一个会等比缩放
    6. border设置图像边框粗细
  7. 超链接标签： <a href="跳转目标" target="目标窗口的弹出方式">文本或图像</a><br> target（_self为默认值，_blank为在新窗口中打开方式）
    1. 外部链接
    2. 内部链接（网页之间的跳转）
    3. 空链接（<a href="#">文本</a>）
    4. 下载链接（地址链接的是.exe或.zip压缩包）
    5. 网页元素链接
    6. 锚点链接<a href="#two">文本</a> <h3 id="two">文本</h3>
  8. 注释标签
    <!-- -->  "ctrl+ /"
  9. 特殊字符
    1. 空格符 &nbsp;
    2. < &lt;
    3. > &gt;
    4. & &amp;
  10. 表格标签
    1. 表格结构：
      1. <table></table>定义表格的标签
      2. <tr></tr>定义表格中的行
      3. <td></td>表格中的单元格
      4. <th></th>表头单元格，加粗居中
    2. 表格属性：
      1. align：left,center,right  对齐方式
      2. width
      3. height
      4. border：1或""  规定是否拥有边框，以及线的宽度
      5. cellspacing：规定单元格之间的空白，默认2像素
      6. cellpadding：规定单元边沿与其内容之间的空白，默认1像素
    3. 结构标签：
      1. <thread></thread>：用于定义表格的头部，内部必须拥有<tr>标签
      2. <tbody></tbody>：用于定义表格的主体
    4. 合并单元格：
      1. 跨行合并：rowspan="合并单元格的个数"，最上侧单元格为目标单元格，写合并代码
      2. 跨列合并：colspan="合并单元格的个数"，最左侧单元格为目标单元格，写合并代码
      3. 注意合并单元格之后删除多余的单元格
  11. 列表标签
    1. 无序列表
      1. <ul>表示html页面中项目的无序列表，其中只能嵌套<li></li>
      2. <li>表示列表项，<li></li>之间相当于容器，可以容纳所有元素
    2. 有序列表
      1. <ol>表示html页面中项目的有序列表，其中只能嵌套<li></li>
      2. <li>与无序列表中相同
    3. 自定义列表:常用于对名词进行解释和描述
      1. <dl>表示html页面中项目的定义描述列表，其中只能嵌套<li></li>，但对他们的个数没有限制
      2. <dt>定义项目或名字
      3. <dd>描述每一个项目或名字
  12. 表单标签
    1. 表单域
      1. 表单域是一个包含表单元素的区域
      2. <form>用于定义表单域
      3. <form>会把它范围内的表单元素信息提交给服务器
      4. 属性
        1. action用于指定接收并处理表单数据的服务器程序的url地址
        2. method用于设置表单数据的提交方式，其取值为get或post
        3. name用于指定表单的名称，以区分同一个页面中的多个表单域
    2. 表单控件（表单元素）
      1. <input>输入表单元素
        1. type="属性值"
          1. button 定义可点击按钮（多数情况用于通过js启动脚本)
          2. checkbox 定义复选框
          3. file 定义输入字段和浏览按钮，供文件上传
          4. hidden 定义隐藏的输入字段
          5. image 定义图像形式的提交按钮
          6. password 定义密码字段。该字段的字符被掩码
          7. radio 定义单选按钮
          8. reset 定义重置按钮。清除表单中的所有数据
          9. submit 定义提交按钮。提交按钮会把表单数据发送到服务器
          10. text 定义单行的输入字段，用户可在其中输入文本，默认宽度为20个字符
        2. name="名称"：定义input元素的名称
        3. value=""：规定input元素的值（未输入元素时显示的字样）
        4. checked="checked"：规定此inout元素首次加载时应当被选中
        5. maxlength="正整数"：规定输入字段中的字符的最大长度
        6. id="":对表单对应的标注标签有指引作用
        7. <label>为input元素定义标注
          1. 用于绑定一个表单元素，当点击<label>内的文本时，浏览器就会自动将焦点（光标）转到或者选择对应的表单元素上，用来增加用户体验
          2. 语法：<label for="sex">女</label>   注：sex为对应input的id
      2. select下拉表单元素（有多个选择，节约页面空间）
        1. <select></select>包含option
        2. >option>>/option>
        3. 在<option>中定义selected="selected"时，当前项即为默认选中项
      3. textarea文本域元素
        1. <textarea>用于定义多行文本输入的控件（常见于留言板，评论）
        2. 语法：<textarea rows="显示的行数" cols="每行中的字符数">文本内容（默认内容）<textarea>（实际开发一般用CSS改变大小）
    3. 提示信息

四、路径

  1. 相对路径（图片相对于html页面的位置）
    1. 同一级路径 <img src="baidu.gif" />
    2. 下一级路径 <img src="images/baidu.gif" />
    3. 上一级路径 <img src="../baidu.gif" />
  2. 绝对路径（E:\或者网址）
